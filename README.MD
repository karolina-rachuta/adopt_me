## Dictionary

- CDN (Content Delivery Network) - server assets (js, css, images):
  (1)szybciej dziaa, bo uzytkownich ma w cachu zapamietana reactowa aplikacje - CDN z cache'
  (2) CDN robi assety w zonach na swiecie, przez co przesyl danych dziala tak samo w roznych czesciach swiata
- Transpilator (i.e. Babel Typescript) === transformation and compilation. Conversion between languages version.
- SPA - single page application (brak przeadowa strony, przypomina aplikacje desktopowe)
- PWA - progressive web apps - to SPA, ale dziaa bez internetu, np. starbucks - apka gdzie mozna zam贸wi i zapaci za ni bez interentu
- SSR - server side rendering (piszemy aplikacj, ale nie przegldarka renderuje t stron tylko robi to server, dziaa szybciej niz normalna aplikacja plus wtedy google boty widz ten HTML i mog go indeksowa)
- Memoizing - cache function result instead call it twice with the same parameters, only for pure functions and limited combination of parameters value
